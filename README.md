## 🌍 Problem Statement

Millions of tons of reusable and recyclable materials end up in landfills every year due to:

- Lack of awareness about waste value  
- Fragmented recycling systems  
- Difficulty sourcing recyclable materials  
- No unified circular economy platform  

### Impact of the Problem
- Environmental pollution  
- Increased CO₂ emissions  
- Loss of valuable raw materials  
- Missed income opportunities  
- Slow adoption of circular economy practices  

---

## 💡 Our Solution

ReLoop AI transforms waste into measurable economic and environmental value.

Using AI-powered image analysis and smart matching, the platform:
- Identifies waste materials
- Generates a Digital Resource Passport
- Estimates market value and CO₂ savings
- Connects users with recyclers, NGOs, and industries
- Enables a circular marketplace

> “Every discarded item deserves a second life.”

---

## 🚀 Key Features

### 🤖 AI Resource Scanner
- Upload waste images
- Identifies material type, quantity, condition, recyclability

### 📄 Digital Resource Passport
- Material composition
- Circularity score
- Market value estimation
- Environmental impact (CO₂ saved)
- Recommended action

### 🏪 Circular Marketplace
- Sell recyclable materials
- Donate reusable items
- Exchange resources
- Connect with verified buyers

### 🎯 Smart Matching Engine
Matches users with recyclers, NGOs, scrap dealers, and industries based on:
- Material type
- Quantity
- Location
- Demand

### 🏘️ Community Collection Pools
- Apartments
- Schools and colleges
- Offices and institutions

### 🌱 Sustainability Dashboard
- Waste diverted from landfills
- CO₂ emissions saved
- Materials recycled
- Revenue generated

### 🏆 Circularity Score
Gamified score encouraging sustainable behavior.

## ▶️ Run Locally

### Prerequisites
- Node.js

Install dependencies:

npm install


Create `.env.local` file:

GEMINI_API_KEY=your_api_key_here


Run the app:
 npm run dev
